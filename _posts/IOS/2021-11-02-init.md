## designated initializer //기본형태
~~~
class MyInfo {
  var name: String
  var myId: String
  
  //
  init(name: Stirng, id: String) {
      self.name = n
      self.myId = id
  }
}

var myInfo1 = MyInfo(m: "kim", id: "abcd")

myInfo1.myId
myInfo1.name
~~~
출력
~~~
"abcd"
"kim"
~~~
init을 사용하면, 처음에 생성되는 시점에, 받아서 세팅을 해줘야한다.  
위코드를 보면 myInfo1을 생성할때 값을 받아야함  
~~~
위에 값에 
  var name: String = ""
~~~
이라고 세팅해줘도 되고 아니면
~~~
  init() {
      self.name = ""
      self.myId = ""
  }
~~~
이렇게 세팅해 줘도 된다.
