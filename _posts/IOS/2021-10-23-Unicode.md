## Unicode(범위관련)
### 숫자만 입력받을 수 있어야 한다.  

~~~
"\u{30}" == "0" //true
"\u{39}" == "9" //ture
~~~
예를 들어, 

~~~
let inputValue = "7"

if inputValue >= "\u{30}" && inputValue <= "\u{39}" {
  print("숫자다")
} else{
  print("숫자가 아니다")
}
~~~  
이면 출력값은
~~~
숫자다 
~~~
가 나오게 되고,  
~~~
let inputValue = "ㅎ"

if inputValue >= "\u{30}" && inputValue <= "\u{39}" {
  print("숫자다")
} else{
  print("숫자가 아니다")
}
~~~  
이면 출력값은
~~~
숫자가 아니다 
~~~
가 나오게 된다.  
  
### 영어만 입력 받아야 한다.
~~~
"\u{41}" == "A" //true
"\u{7a}" == "z" //ture
~~~
하나하나만 인식 가능하다.  
7a는 16진수여서 그런것이다.
