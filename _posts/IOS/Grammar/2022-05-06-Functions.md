# Functions
- 불필요한 반복을 줄여준다.

##  Calling Functions

~~~
print("hello")
~~~
출력
~~~
hello
~~~

## Defining Functions
- 파라미터는 생략할 수 있다.

~~~
func sayHello() {
  print("hello, swift")
}

sayHello()
~~~
출력
~~~
hello, swift
~~~
