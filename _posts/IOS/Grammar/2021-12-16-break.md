## break  

ex)
~~~
for row in 0...5 {
  if row > 2{
    break
  }
  print("\(row) was executed!")
}
~~~  

실행결과
~~~
0 was executed!
1 was executed!
2 was executed!
~~~  

원래대로라면 0~5까지의 값만큼 6회에 걸쳐서 반복 실행하야된다.  
하지만 내부 실행 블록에서는 루프 상수를 비교하여 2보다 큰 값일 경우 break 구문을 실행하도록  
조건문이 작성되어 있어서 row가 2보다 커지는 순간 break구문이 실행되어 종료된것이다.
